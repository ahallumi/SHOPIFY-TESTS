# SHOPIFY-TESTS

## Shopify section included

- `sections/mix-match-bundle.liquid` — product section for a Mix & Match bundle picker.

## How to use this in Shopify

1. Copy `sections/mix-match-bundle.liquid` into your theme's `sections/` folder.
2. In Shopify Admin, go to **Online Store → Themes → Edit code** and upload/paste the file if needed.
3. Open the Theme Editor on a product template and add the **Mix & Match bundle** section.
4. Configure the tier settings, colors, button labels, and prompt text in the section settings.
